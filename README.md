# Portafolio Profesional

Bienvenido al repositorio de mi portafolio profesional. Este proyecto ha sido diseñado y desarrollado desde cero para demostrar mis capacidades como **Full Stack Developer**, priorizando el rendimiento, la escalabilidad y una arquitectura limpia en el frontend.

## Stack Tecnológico

Elegí una arquitectura moderna y ligera, evitando la sobrecarga de dependencias innecesarias:

- **Core:** React 18 + Vite (SPA)
- **Lenguajes:** JavaScript (ES6+), HTML5 Semántico
- **Estilos:** CSS3 Nativo (Arquitectura modular y responsive sin frameworks pesados como Bootstrap)
- **Routing:** React Router v6
- **Features:**
  - **i18n:** Internacionalización completa (Español/Inglés) con `i18next`
  - **Temas:** Sistema de Modo Oscuro/Claro persistente con Context API
  - **Animaciones:** `tsparticles` y transiciones CSS optimizadas

## Puntos Clave

- **Arquitectura Escalable:** Estructura de carpetas basada en funcionalidades y componentes reutilizables.
- **Responsive Design:** Diseño fluido 100% adaptativo (Mobile-First), con menús táctiles y layouts dinámicos para cualquier tamaño de pantalla.
- **Rendimiento:** Optimización de carga, code-splitting y uso eficiente de recursos.
- **UX/UI:** Interfaz limpia, profesional y accesible.

## Ejecución Local

Si deseas explorar el código fuente o probar las funcionalidades en local:

```bash
# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev

# Generar build de producción
npm run build
```
